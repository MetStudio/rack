Computes sizes of segments having intensity over threshold.

Parameters
threshold- threshold reflectivity
area- fuzzy threshold, at which the likelihood is 50%
polar- use bin geometry; else scale to Cartesian bin area

This operator is universal , it is computed on DBZ but it applies also to str radar parameters measured (VRAD etc)
