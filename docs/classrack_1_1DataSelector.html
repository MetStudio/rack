Detailed Description

Tool for selecting datasets based on paths, quantities and min/max elevations.

Future version will use:

  • data=
  • dataset=

Applies drain::RegExp in matching.

Detect the last path accepted by regexp and increment the trailing numeral (eg. data2 => data3) and return the std::string in path.

Parameters
src- hdf tree to be searched
pathRegExp- parent path(s) defined as regular expressions, for example:
  • /dataset[0-9]$
  • /dataset2/data[0-9]$
  • data[0-9]$
  • data2/quality[0-9]$
path- std::string in which the result is stored, if found.
Returns
- true, if a path was found

Notice that path can be initialized with a default value; if no path is found, the default value will stay intact.
