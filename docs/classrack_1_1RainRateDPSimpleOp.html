Detailed Description

Uses DBZ, ZDR and KDP to derive rain rate. Under construction.

RATE_KDP = sign(KDP) * kdp_aa * (|KDP| ** kdp_bb).

where

kdp_aa = 40.6 kdp_bb = 0.866
