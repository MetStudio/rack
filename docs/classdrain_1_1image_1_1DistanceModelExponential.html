Detailed Description

In this model, the diagonal decay is computed as $ d_d = - { ^2 d_x + ^2 d_x)) } ; $ because $ d_d = - { ^2 d_x + ^2 d_x)) } ; $

Sets the geometry of the distance model.

The values are in float, as eg. half-width radius may be sharp, under 1.0.

Parameters
horz- horizontal distance, "width".
vert- vertical distance, "height"; if negative, set equal to horizontal

Special values:

  • NaN: use default value (horz for vert and horzRight; vert for )
  • negative value: no decay; continue to infinity with constant value
  • zero: sharp decay, value is zero in neighboring pixels
