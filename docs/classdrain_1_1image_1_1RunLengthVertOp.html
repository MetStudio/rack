Detailed Description

Computes lengths of vertical segments.

drainage shapes1.png --iRunLengthVert 128 -o runLengthVertAbs.png
drainage shapes1.png --physicalRange 0:1 --iRunLengthVert 0.5 -o runLengthVert.png

Computes lengths of horizontal or vertical segments.

drainage image.png --iRunLengthHorz 128 -o out.png

To detect line segments longer than 255 the user should use 16-bit result image, –type 16 . Further, –scale command may be needed to make segment lengths visible in 16-bit scale.
