Detailed Description

template<class S, class D>
class drain::image::SegmentProber< S, D >

A helper class applied by FloodFillOp and SegmentAreaOp.

Template Parameters
S- storage type of the source image (int by default, but should be floating-type, if src is).
D- storage type of the destination image, the image to be filled.

A semi-recursive approach that turns the 2D recursion to 1D traversal + 1D recursion.

the horizontal direction is handled sequentially whereas the vertical direction handled recursively. Compared to fully recursive approach, this technique preserves the speed but offers smaller consumption of memory.
