Detailed Description

template<typename DT>
class rack::PlainData< DT >

The template parameters are expected to be exactly the following, but all of them const or non-const:

Template Parameters
T- HDF5 tree, as HI5TREE
D- two-dimensional data, as drain::image::Image
M- metadata, as rack::PolarODIM
